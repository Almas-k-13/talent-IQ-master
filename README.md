# 🚀 TalentIQ – Realtime Coding Interview Platform

A premium full-stack web application built using the MERN stack with modern UI/UX, authentication system, collaborative interview rooms, live code editor, and integrated video calling.

# 📌 Project Overview

TalentIQ provides a complete environment for conducting technical interviews:

* Secure Authentication System (Clerk)
* Coding Problems with difficulty levels
* Realtime Interview Rooms
* Video Call + Live Code Editor
* Session Management Dashboard
* Collaborative Developer Experience

---

# ✨ Major Features

## 🔐 Authentication & Security
* Clerk Authentication (Test keys for local, Live keys for production)
* Protected Routes
* Secure JWT Verification
* Persistent Login Sessions

---

## 👥 Interview Room Functionalities
* Create Session with coding problem
* Invite Participants
* Video Call Integration
* Collaborative Code Editor
* Session Summary & End Flow

---

## 📊 Dashboard
* Active Sessions
* Recent Sessions
* Session by ID with auto-refresh
* Session Management (Join / End)

---

# ⚡ Tech Stack

## Frontend Technologies
* React.js (Vite)
* Tailwind CSS
* Axios
* React Query
* Clerk React SDK

## Backend Technologies
* Node.js
* Express.js
* MongoDB + Mongoose
* JWT Authentication
* Clerk Middleware
* Socket.IO
